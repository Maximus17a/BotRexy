# Directorio para imágenes estáticas

Este directorio contiene las imágenes estáticas utilizadas en la aplicación web.

## Subdirectorios

### `/backgrounds`
Contiene las imágenes de fondo personalizadas que los usuarios suben para sus mensajes de bienvenida.

**Formatos soportados:**
- PNG
- JPG/JPEG
- GIF
- WebP

**Características:**
- Las imágenes se redimensionan automáticamente a 800x300px
- Los nombres de archivo incluyen el ID del servidor para evitar colisiones
- Los archivos se eliminan automáticamente cuando el usuario remueve la imagen de fondo
